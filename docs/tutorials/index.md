# Tutorials

Welcome to the TAPPaaS tutorials! These hands-on guides will walk you through common tasks and workflows, helping you learn by doing. Each tutorial includes clear instructions, code examples, and expected outcomes.

## Learning Paths

### Beginner Path
Start here if you're new to TAPPaaS. These tutorials will get you up and running quickly.

1. [Get Started](/tutorials/get-started/) - Learn the fundamentals
2. [Deploy Your First App](/tutorials/get-started/deploy-first-app/) - Deploy a sample application
3. [Configure Application](/tutorials/get-started/configure/) - Customize your deployment

**Estimated Time**: 1-2 hours

### Intermediate Path
Build on the basics with language-specific deployments and integrations.

1. [Deploy a Python Application](/tutorials/deployment/python/) - Flask, Django, FastAPI
2. [Deploy a Node.js Application](/tutorials/deployment/nodejs/) - Express, Next.js, NestJS
3. [Database Integration](/tutorials/integration/databases/) - PostgreSQL, MySQL, MongoDB
4. [Custom Domain Setup](/tutorials/integration/domains/) - DNS and SSL/TLS

**Estimated Time**: 3-4 hours

### Advanced Path
Master advanced features for production deployments.

1. [Multi-Tenancy Setup](/tutorials/operations/multi-tenancy/) - Isolate applications by tenant
2. [Backup and Recovery](/tutorials/operations/backup/) - Protect your data
3. [CI/CD Integration](/tutorials/integration/cicd/) - Automate deployments
4. [Monitoring and Observability](/tutorials/operations/monitoring/) - Track performance

**Estimated Time**: 4-6 hours

## Prerequisites Overview

Most tutorials assume you have:

- A Kubernetes cluster (version 1.24+)
- `kubectl` CLI tool installed and configured
- Basic familiarity with Kubernetes concepts
- Command line experience

Specific tutorials may require additional tools or knowledge, which will be listed in their individual prerequisites sections.

## How to Use These Tutorials
